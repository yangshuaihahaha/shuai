# 热门实时商品统计
基本需求
    - 统计近1小时内的热门商品，每5分钟更新一次
    - 热门度用浏览次数（"pv"）来衡量
解决思路
    - 在所有用户行为数据中，过滤出浏览（"pv"）行为进行统计
    - 构建滑动窗口，窗口长度为1小时，滑动距离为5分钟
